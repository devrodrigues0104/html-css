Formatações

Negrito/Destaque

Utilizando a tag <b>, temos o negrito não semântico, diferente da tag <strong>, que faz o negrito semantico.

Itálico/Enfase

Utilizando a tag <i>, temos o itálico não semântico, diferente da tag <EM>, que faz o itálico semântico.

Texto marcado

Para marcar um texto, posso usar a tag <mark>.

Grande e Pequeno

Usamos <small> para textos pequenos e <big>, apesar de ser um termo obsoleto, para textos grandes.

Texto deletado e texto inserido

Podemos marcar um texto com <del> como texto deletado para indicar que ele deve ser lido, porém não considerado.

Podemos marcar um texto com <ins> como texto inserido para dar ênfase e indicar que ele foi adicionado depois. Também a existe a tag <u> para sublinhar, porém não possui um valor semântico.

Texto Subscrito e sobrescrito

<sub> para H2O e <sup> para x2-2=0.
