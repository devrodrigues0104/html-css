Representação de cores no CSS3

As cores no CSS podem ser representadas de três maneiras: #HEX, RGBa e HSLa. HSL segue o padrão hsla(n, n%, 100%, n), respectivamente matiz (0 a 360), saturação (0% a 100%), luminosidade (0 a 100) e transparência (0 a 1).

Para o exemplo do azul temos: #0000ff, rgb(0, 0, 255), hsl(240, 100%, 50%).

Formatação de fontes

Medidas de fontes

  • Medidas absolutas
    • px - recomendada pela w3c
    • cm
    • mm
    • pt
    • pc
  • Medidas relativas
    • em - recomendada pela w3c
    • ex
    • rem
    • vw
    • vh
    • %

Exemplo 1

Fontes podem ser formatadas utilizando font-weight:, font-size:, font-style: e font-family:.

            
    h1{
        font-family: cursive, "Comic Sans MS";
        font-size: 1.125em;
        font-style: italic;
        font-weight: bolder;
    }

Exemplo 2 (shorthand)

Além desta forma, as fontes podem ser formatadas utilizando font: na seguinte ordem de parâmetros:

  1. font-style
  2. font-weight
  3. font-size
  4. font-family
            
    h1{
        font: italic bolder 1.125em cursive, "Comic Sans MS";
    }

OBS: É possível deixar de colocar um parâmetro desde de que os outros estejam na ordem certa

Baixando fontes

Essa fonte é a Holy Friday. Ela foi baixada pelo site DaFont.

Para usar fontes baixadas é preciso criar um seletor para isso, o @font-face. Para usar o font face o tipo do arquivo baixado deve ser declarado.

Tipos de arquivo de fonte
  • opentype (otf)
  • truetype (ttf)
  • embedded-opentype
  • truetype-aat
  • svg
